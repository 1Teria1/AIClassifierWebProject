# AI Support Classifier

> Автоматическая классификация обращений в службу поддержки на основе RuBERT

## 📋 О проекте

**AI Support Classifier** — это веб-сервис для автоматической сортировки обращений в службу поддержки. Сервис анализирует текст сообщения и определяет:

- **Категорию обращения** (Оплата, Доставка, Техническая проблема, Вопрос по товару, Спам)
- **Приоритет** (Высокий, Средний, Низкий)
- **Уверенность модели** (confidence score)

### Зачем это нужно?

В крупных компаниях ежедневно поступают тысячи обращений. Ручная сортировка занимает часы работы сотрудников. Наш сервис автоматизирует этот процесс, сокращая время реакции на критичные запросы в 5-10 раз.

### 🧠 Используемые технологии

| Компонент | Технология |
|-----------|------------|
| ML-модель | RuBERT |
| Фреймворк | PyTorch |
| API | RestAPI на Flask |
| Интерфейс | HTML/CSS/JavaScript |
